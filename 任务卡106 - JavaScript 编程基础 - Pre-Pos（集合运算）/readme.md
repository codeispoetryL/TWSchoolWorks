# JavaScript 集合操作基础练习

## 需求说明
1. 两个集合比较筛选 - 其中包含4个小练习
2. 单个集合内各种元素统计 - 其中包含3个小练习
3. 两个集合比较+筛选+统计 - 其中包含4个小练习

## 环境
JavaScript ES6。

## 开始
在命令行中使用以下命令在用户本地任意目录下clone此题目库
```
git clone repo_of_this_template
```
首先初次下载完需要安装依赖：
```
npm install
```
运行所有测试：
```
npm test
```

用任意编辑器打开clone下来的文件夹，内部会存在两个文件夹
```
spec  //测试文件夹
src   //源文件
```
